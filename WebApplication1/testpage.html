区块二

首先从这里下载书的源代码。以防链接无效，就Google一下“professional javascript for web developers 3rd edition source code”。下载“Full Code Download”，大约 3.5 MB。文件名是“9781118026694_download.zip”。解压文件然后你得到“9781118026694_download”。重命名文件夹为professional-javascript。文件夹中包含一系列子文件夹（例如，Ch03，Cho4 等等）。

区块三
